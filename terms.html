Términos y Condiciones de SmarfTech

Última actualización: 25 de junio de 2026

1. Objeto

SmarfTech ofrece soluciones tecnológicas relacionadas con automatización, integración de sistemas, asistentes virtuales, inteligencia artificial y atención automatizada mediante canales de mensajería como WhatsApp.

2. Aceptación de los términos

Al utilizar los servicios, aplicaciones o sistemas de SmarfTech, el usuario o empresa contratante acepta estos Términos y Condiciones.

```

Si no está de acuerdo con ellos, debe abstenerse de utilizar los servicios.

```

3. Naturaleza del servicio

SmarfTech puede proporcionar sistemas que reciben mensajes, procesan consultas, generan respuestas automáticas, consultan información empresarial y ejecutan integraciones con herramientas de terceros.

```

Las características disponibles pueden variar según el plan, configuración, alcance contratado y etapa de desarrollo del servicio.

```

4. Uso de inteligencia artificial

Algunos servicios pueden utilizar modelos de inteligencia artificial para interpretar mensajes, generar respuestas, resumir información o ejecutar tareas.

```

Las respuestas generadas por inteligencia artificial pueden contener errores, omisiones, interpretaciones incorrectas o información desactualizada.

SmarfTech no garantiza que toda respuesta automática sea exacta, completa o adecuada para decisiones médicas, legales, financieras, de seguridad u otras decisiones de alto riesgo.

```

5. Responsabilidades de la empresa cliente

La empresa que contrata SmarfTech es responsable de:

```
  • Proporcionar información correcta y actualizada.
  • Definir qué respuestas y procesos desea automatizar.
  • Revisar periódicamente el funcionamiento del sistema.
  • Informar a sus clientes cuando corresponda sobre el uso de automatización o IA.
  • Cumplir la normativa aplicable en materia de privacidad y protección de datos.
  • No utilizar el servicio para spam, fraude o actividades prohibidas.
```

6. Uso permitido

El servicio debe utilizarse únicamente para fines legales y legítimos.

```

Está prohibido utilizar SmarfTech para:

  • Enviar spam o comunicaciones no autorizadas.
  • Suplantar personas o empresas.
  • Realizar fraudes, estafas o actividades engañosas.
  • Distribuir contenido ilegal, dañino o malicioso.
  • Intentar vulnerar sistemas, credenciales o información de terceros.
  • Infringir las políticas de Meta, WhatsApp u otros proveedores.
```

7. Servicios de terceros

SmarfTech puede depender de servicios de terceros como Meta, WhatsApp, proveedores de inteligencia artificial, servicios de alojamiento, bases de datos, APIs y plataformas de automatización.

```

La disponibilidad y funcionamiento de estas plataformas dependen de sus propios términos, políticas, límites y condiciones técnicas.

```

8. Disponibilidad del servicio

SmarfTech procurará mantener sus servicios disponibles y funcionando correctamente.

```

Sin embargo, no se garantiza disponibilidad ininterrumpida o libre de errores. Pueden producirse interrupciones por mantenimiento, fallas técnicas, problemas de Internet, cambios de APIs o incidencias de proveedores externos.

```

9. Modificaciones y actualizaciones

SmarfTech puede modificar, actualizar, reemplazar o retirar funcionalidades cuando sea necesario por razones técnicas, comerciales, legales o de seguridad.

10. Propiedad intelectual

El software, código, diseños, procesos, documentación, marcas y contenidos desarrollados por SmarfTech pertenecen a sus respectivos titulares.

```

La contratación de un servicio no implica la transferencia de propiedad del código fuente o de la plataforma, salvo acuerdo escrito en contrario.

```

11. Datos y confidencialidad

SmarfTech tratará la información de acuerdo con su Política de Privacidad y con los acuerdos establecidos con cada empresa cliente.

```

La empresa contratante debe proteger sus credenciales, accesos, tokens y cuentas empresariales.

```

12. Limitación de responsabilidad

En la medida permitida por la ley, SmarfTech no será responsable por pérdidas indirectas, interrupciones comerciales, decisiones tomadas exclusivamente con base en respuestas automáticas o daños derivados de información incorrecta proporcionada por la empresa cliente.

```

Tampoco será responsable por fallas atribuibles a Meta, WhatsApp, proveedores de inteligencia artificial, servicios de Internet u otras plataformas externas.

```

13. Suspensión o terminación

SmarfTech puede suspender o terminar el acceso al servicio cuando exista:

```
  • Incumplimiento de estos términos.
  • Uso ilegal, abusivo o fraudulento.
  • Riesgo para la seguridad del sistema.
  • Falta de pago, cuando corresponda.
  • Incumplimiento de políticas de proveedores externos.
```

14. Cambios en los términos

SmarfTech puede actualizar estos Términos y Condiciones cuando cambien sus servicios, tecnologías, requisitos legales o prácticas comerciales.

```

La versión vigente estará disponible en esta misma página.

```
